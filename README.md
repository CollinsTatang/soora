## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- React Native

### Setup

```bash
git clone https://github.com/CollinsTatang/soora.git
cd ./soora
```

### Install

```bash
 install expo
```

### Usage

```bash
expo start
```

### Run tests

```bash
npm test
```
